About Device Registration

Registering a device adds it to the system inventory. Once registered, you can:

  • Assign it to an access point
  • Monitor its status
  • Configure its settings
  • View its access logs
Tip: The API key is used by the device to authenticate with the server. Keep it secure!
Device Types Explained
Fingerprint Scanner: Biometric verification using fingerprints
Facial Recognition: Camera-based facial verification
Turnstile / Gate: Physical barrier/access gate
Smart Lock: Electronic door lock (remote controlled)
Keypad: PIN/Code entry device (for visitors/emergency)
Other: Any other hardware device
Typical Access Point Setup:
  • Primary: Fingerprint Scanner + Turnstile
  • Primary: Facial Recognition + Smart Lock
  • Secondary: Keypad (visitors/emergency) + Smart Lock
