# Deployment Guide

## Requirements
- Python 3.10+
- Fingerprint scanner (R307)
- HD Webcam
- PostgreSQL (production) / SQLite (prototype)

## Steps
1. Clone the project repository
2. Create and activate virtual environment
3. Install dependencies using requirements.txt
4. Configure environment variables
5. Initialize database
6. Run the application using main.py

## Hardware Integration
- Connect fingerprint scanner via USB/Serial
- Connect webcam
- Configure access relay via Arduino
