# Project Structure Documentation

This document explains the purpose of each file and folder in the project root.

## Root Files
- `main.py` – Entry point of the biometric access control system
- `README.md` – Project overview, setup, and usage instructions
- `requirements.txt` – Python dependencies
- `setup.py` – Packaging and installation configuration
- `.env` – Environment variables (excluded from version control)
- `.env.example` – Sample environment configuration
- `.gitignore` – Files excluded from version control
- `LICENSE` – Open-source license

## Folders
- `src/` – Core application logic and system modules
- `ui/` – User interface components for enrollment and verification
- `scripts/` – Database initialization, reporting, and admin utilities
- `tests/` – Unit, integration, and system tests
- `docs/` – System documentation and diagrams
- `data/` – Database and persistent storage
